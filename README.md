## How to Contribute
1. Install the dependencies
```
pip install -r requirements.txt
```
2. Modify the code
3. Test building an executable package with `pyinstaller`
```
pyinstaller --noconfirm .\tomato-timer.spec
````
4. Generate and zip the package
```
create-release.ps1
```
